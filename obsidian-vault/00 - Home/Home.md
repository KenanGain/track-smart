## How to use this vault

1. Open this folder in Obsidian (`File → Open Vault → Select Folder`).
2. Start each work session from a [[Daily Note]] (`Ctrl+P → Daily notes: Open today's daily note`).
3. Capture every meaningful decision in [[Decisions Index]].
4. Track work in [[Active Tasks]].
5. Use the **Graph view** (`Ctrl+G`) to see how pages, data, and features connect.

## Search & Navigation tips

- `Ctrl+O` → Quick switcher (jump to any note)
- `Ctrl+Shift+F` → Global search
- `Ctrl+G` → Graph view
- `Ctrl+P` → Command palette
- `[[` → Insert wiki-link to another note
