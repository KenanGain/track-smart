# Daily Notes — README

This folder is the destination for daily notes.

- **Format:** `YYYY-MM-DD.md` (e.g. `2026-04-26.md`)
- **Template:** `templates/Daily Note.md`
- **Open today's note:** `Ctrl+P → Daily notes: Open today's daily note`

This README is just a placeholder so the folder isn't empty. You can delete it once you've created your first daily note.
